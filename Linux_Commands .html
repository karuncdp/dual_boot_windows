    🐧 Linux Command Reference

    A comprehensive, production-quality reference guide to Linux commands.
    Last Updated: July 2026

    Welcome to this comprehensive guide to Linux commands.

    Whether you're just beginning your journey with the Linux command line or you're already comfortable working in a Linux environment, this guide is designed to be a practical reference. Beginners can use it to build a solid foundation, while experienced users can use it as a quick command handbook and refresher.

    The primary focus is on Debian and Ubuntu-based systems. However, the vast majority of the commands discussed are part of standard Linux utilities and work across most Linux distributions. The primary differences between distributions usually lie in package management and a few system-specific tools. For example, Debian-based distributions use apt, while Fedora uses dnf, Arch Linux uses pacman, and openSUSE uses zypper.

    The best way to learn Linux is by using it—experiment with every command in a terminal as you read through this guide. And you don't have to remember each commands at all, that is not the way of learning, you use it for many times and it comes to you naturally afterwards so don't worry if you don't remember the commands at the beginning. You can easily refer to this guide any time you feel you need it.

    1. Basic System Commands

    whoami

    Prints the current username.

    Syntax:

    whoami

    Example:

    whoami

    Output:

    ubuntu

    pwd

    Displays the current working directory (Print Working Directory).

    Syntax:

    pwd

    Example:

    pwd

    Output:

    /home/ubuntu/projects

    cd

    Changes the current directory.

    Syntax:

    cd <directory>

    Special Paths:

    Path Meaning
    / Root directory (top level)
    ~ Home directory
    .. One level up (parent directory)
    . Current directory

    Examples:

    cd /var/log          # Navigate to /var/log
cd ~                 # Go to home directory
cd ..                # Go one level up
cd ../..             # Go two levels up
cd ~/Documents       # Go to Documents in home

    ls

    Lists all files and directories in the specified location.

    Syntax:

    ls [flags] [directory]

    Relevant Flags:

    Flag Description
    -a List all files, including hidden files (those starting with .)
    -l Long format — shows permissions, owner, size, modification date
    -h Human-readable sizes (e.g., 2.5M, 1.2G) — use with -l
    -i Display inode (index node) numbers
    -d */ List only subdirectories

    Examples:

    ls

    Output:

    Documents  Downloads  file.txt  script.py

    ls -la

    Output:

    drwxr-xr-x 5 ubuntu ubuntu 4096 Jul 09 10:00 .
drwxr-xr-x 3 root   root   4096 Jul 08 09:00 ..
-rw-r--r-- 1 ubuntu ubuntu  220 Jul 09 08:00 .bash_logout
-rw-r--r-- 1 ubuntu ubuntu 3771 Jul 09 08:00 .bashrc
-rw-r--r-- 1 ubuntu ubuntu  898 Jul 09 09:30 file.txt

    ls *.py

    Output:

    script.py  utils.py  main.py

    ls -d */

    Output:

    Documents/  Downloads/  Projects/

    clear

    Clears the terminal screen.

    Syntax:

    clear

    history

    Displays all previously executed commands in the current session and past sessions.

    Syntax:

    history

    Pro Tips:

    history | grep "ssh"     # Search for past SSH commands
! 42                     # Execute command number 42 from history

    man

    Displays the manual (documentation) for a given command.

    Syntax:

    man <command>

    Navigation: - Press q to quit the manual viewer. - Use ↑/↓ or Page Up/Page Down to scroll.

    Example:

    man ls

    shutdown

    Shuts down or reboots the system.

    Syntax:

    shutdown [options]

    Relevant Options:

    Option Description
    -h Halt (shutdown) after 1 minute (default)
    -h now Shutdown immediately
    -h +10 Shutdown after 10 minutes
    -c Cancel a scheduled shutdown
    -r now Reboot immediately

    Examples:

    sudo shutdown -h now      # Immediate shutdown
sudo shutdown -h +10      # Shutdown in 10 minutes
sudo shutdown -c        # Cancel scheduled shutdown
sudo shutdown -r now    # Reboot now

    💡 Tip: Create an alias for convenience:

    alias sd='sudo shutdown -h now'

    Note: Aliases persist only for the current session unless added to ~/.bashrc.

    alias

    Creates a shortcut (alias) for a command.

    Syntax:

    alias <name>='<command>'

    Example:

    alias lst='tail -50'
alias gs='git status'

    To make aliases permanent:

    nano ~/.bashrc
# Add aliases at the end of the file, then save and exit
source ~/.bashrc   # Reload the configuration

    open

    Opens a file or program with its default application.

    Syntax:

    open <file_or_program>

    Examples:

    open hello.pdf        # Open PDF in default viewer
open code             # Open VS Code (if installed)

    Side Note: Update VS Code via terminal:

    sudo apt update && sudo apt upgrade code

    2. Disk & Storage Management

    df

    Reports disk space usage of all mounted filesystems (disk free).

    Syntax:

    df [flags]

    Relevant Flags:

    Flag Description
    -h Human-readable format (e.g., 10G, 500M)

    Example:

    df -h

    Output:

    Filesystem      Size  Used Avail Use% Mounted on
/dev/sda1        50G   20G   28G  42% /
tmpfs           3.9G  1.2M  3.9G   1% /run

    du

    Estimates file and directory space usage (disk usage).

    Syntax:

    du [flags] <path>

    Relevant Flags:

    Flag Description
    -h Human-readable sizes
    -s Summary (total only, not individual files)
    --max-depth=N Show directories up to N levels deep

    Examples:

    du -sh /var/log

    Output:

    1.2G    /var/log

    sudo du -h --max-depth=1 / | sort -h

    Output:

    0       /proc
4.0K    /mnt
120M    /tmp
2.5G    /usr
8.0G    /var

    lsblk

    Lists information about all available block devices (disks, partitions).

    Syntax:

    lsblk

    Example Output:

    NAME   MAJ:MIN RM   SIZE RO TYPE MOUNTPOINT
sda      8:0    0    50G  0 disk
├─sda1   8:1    0    49G  0 part /
└─sda2   8:2    0     1G  0 part [SWAP]
sr0     11:0    1  1024M  0 rom

    Finding Large Files

    List the 20 largest files on the entire system:

    sudo find / -type f -exec du -h {} + 2>/dev/null | sort -rh | head -n 20

    GUI Tools for Disk Analysis

    Tool Installation Description
    ncdu sudo apt install ncdu NCurses Disk Usage — interactive, terminal-based
    baobab sudo apt install baobab GNOME Disk Usage Analyzer — graphical

    3. Encryption & Security

    chmod

    Changes file or directory permissions (change mode).

    Syntax:

    chmod [permissions] <file_or_directory>

    Symbolic Mode Examples:

    chmod +x script.sh        # Make executable
chmod -w datafile.dat     # Remove write permission
chmod 755 myfolder        # rwxr-xr-x
chmod 600 private.key     # rw------- (owner only)

    shred

    Securely overwrites files to prevent recovery (unlike rm, which only marks space as free).

    Syntax:

    shred [flags] <file>

    Relevant Flags:

    Flag Description
    -n N Overwrite N times (default: 3)
    -z Add final overwrite with zeros to hide shredding
    -u Truncate and remove file after overwriting
    -v Verbose mode (show progress)
    -f Force permissions change if needed

    Example:

    shred -n 5 -zu -v secret.txt

    ⚠️ Limitation: Does not work reliably on modern SSDs due to wear-leveling. For SSDs, consider hdparm (use with extreme caution).

    zip (Encrypted)

    Encrypts a folder into a password-protected ZIP archive.

    Syntax:

    zip -er <archive.zip> <folder/>

    Example:

    zip -er my_secure_folder.zip my_folder/
# You will be prompted to enter and verify a password

    To decrypt:

    unzip my_secure_folder.zip
# Enter password when prompted

    ecryptfs

    More secure and robust directory encryption.

    Installation:

    sudo apt install ecryptfs-utils

    Usage:

    sudo mount -t ecryptfs <encrypted_source> <decrypted_destination>

    gpg

    Encrypts and decrypts files using GNU Privacy Guard.

    Encrypt a file:

    gpg -c my_secret.txt
# Creates my_secret.txt.gpg
# You will be prompted for a password

    Decrypt a file:

    gpg -d my_secret.txt.gpg > my_secret.txt

    4. File & Directory Operations

    mkdir

    Creates a new directory (make directory).

    Syntax:

    mkdir <directory_name>
# or
md <directory_name>

    Example:

    mkdir Projects
mkdir -p Projects/web/app   # Create nested directories

    cp

    Copies files and directories.

    Syntax:

    cp [flags] <source> <destination>

    Relevant Flags:

    Flag Description
    -r Recursive (copy directories and their contents)
    -v Verbose (show what's being copied)
    -u Copy only if source is newer than destination
    -n Do not overwrite existing files
    -p Preserve timestamps and permissions

    Examples:

    cp file.txt /backup/
cp -r my_folder/ /backup/my_folder/
cp -v *.log /archive/

    mv

    Moves or renames files and directories.

    Syntax:

    mv <source> <destination>

    Examples:

    mv old_name.txt new_name.txt      # Rename
mv file.txt /home/ubuntu/docs/    # Move
mv folder/ /backup/               # Move directory

    rm

    Removes files and directories permanently.

    Syntax:

    rm [flags] <file_or_directory>

    Relevant Flags:

    Flag Description
    -r Recursive (remove directories and contents)
    -f Force removal without prompting
    -i Interactive (prompt before each removal)

    Examples:

    rm file.txt
rm -r old_project/
rm -rf temp/          # Force remove directory (use with caution!)

    ⚠️ Special Case: If a filename starts with -, rm may interpret it as a flag. Use -- to indicate end of flags:

    rm -- -weird-filename.txt

    find

    Searches for files and directories matching specified criteria.

    Syntax:

    find [path] [options] [expression]

    Common Paths:

    Path Search Scope
    / Entire filesystem (may need sudo)
    ~ Home directory
    . Current directory and subdirectories

    Relevant Options & Expressions:

    Flag/Expression Description
    -maxdepth N Limit search depth to N levels
    -type f Find only regular files
    -type d Find only directories
    -name "pattern" Case-sensitive name match
    -iname "pattern" Case-insensitive name match

    Examples:

    find ~ -type f -name "*.py"

    Output:

    /home/ubuntu/scripts/main.py
/home/ubuntu/utils/helpers.py

    find . -maxdepth 1 -type f

    Output:

    ./file.txt
./script.py
./README.md

    find . -maxdepth 1 -type d

    Output:

    .
./Documents
./Downloads

    5. Git Version Control

    Initial Setup (First Time)

    git init                              # Initialize a new Git repository
git add .                             # Stage all files
git add <file> <folder>               # Stage specific files/folders
git commit -m "Initial commit"        # Commit with a message

# Connect to remote repository
git remote add origin https://github.com/<username>/<repo>.git
# or for GitLab:
git remote add origin https://gitlab.com/<username>/<repo>.git

# If you need to update an existing remote URL
git remote set-url origin https://github.com/<username>/<repo>.git

# Push to main branch
git branch -M main
git push -u origin main

    Daily Workflow

    git add .                             # Stage changes
git commit -m "Updated project with new features"
git push origin main                  # Push to remote

    git rm

    Removes a file from both the working directory and the staging area.

    Syntax:

    git rm <file_to_remove>

    git status

    Shows which files have been modified, staged, or are untracked.

    Syntax:

    git status

    git diff

    Shows the differences between working directory and staging area.

    Syntax:

    git diff

    git log

    Displays commit history.

    Syntax:

    git log

    Example Output:

    commit a3d5f6b91e1b2c8f...
Author: Your Name <you@example.com>
Date:   Sat Oct 12 14:22:18 2025

    Fixed bug in login module

    Relevant Flags:

    Flag Description
    --oneline Compact one-line format per commit
    --graph ASCII graph of branch/merge history
    --all Show all branches

    Example:

    git log --oneline

    Output:

    a3d5f6b Fixed bug in login module
8c2b1a4 Added user dashboard
4f9e2d1 Initial commit

    git checkout

    Switches branches or restores files. Use to temporarily inspect an old commit.

    Syntax:

    git checkout <commit-hash>          # Detached HEAD state — explore old version
git checkout main                   # Return to latest main branch
git checkout <commit-hash> -- path/to/file   # Restore specific file from old commit

    git reset ⚠️

    Permanently reverts to a specific commit. Destructive — use with caution.

    Syntax:

    git reset --hard <commit-hash>

    To push the revert to remote:

    git push origin main --force

    git clone

    Copies a remote repository to your local machine.

    Syntax:

    git clone <repository-url>

    Example:

    git clone https://github.com/torvalds/linux.git

    git pull

    Fetches and integrates the latest changes from the remote repository.

    Syntax:

    git pull

    6. Media Processing

    ffmpeg

    A comprehensive multimedia framework for audio, video, and image processing.

    Key Capabilities: - Format conversion (MP4, MP3, WAV, MKV, PNG, JPG, etc.) - Audio/video extraction and merging - Trimming, resizing, compressing, and applying filters

    Extract Audio from Video

    To MP3 (best quality):

    ffmpeg -i input.mp4 -q:a 0 -map a output.mp3

    To WAV:

    ffmpeg -i input.mp4 -vn -acodec pcm_s16le -ar 44100 -ac 2 output.wav

    Remove Audio from Video

    ffmpeg -i input.mp4 -c copy -an muted.mp4

    Trim Media

    Video (without re-encoding — fast):

    ffmpeg -ss 00:01:00 -to 00:02:30 -i input.mp4 -c copy trimmed.mp4

    Audio:

    ffmpeg -ss 30 -to 90 -i input.mp3 -c copy trimmed.mp3

    Convert Formats

    Video (MP4 → MKV):

    ffmpeg -i input.mp4 -c copy output.mkv

    Audio (WAV → MP3):

    ffmpeg -i input.wav -codec:a libmp3lame -qscale:a 2 output.mp3

    Compress Video

    ffmpeg -i input.mp4 -vcodec libx264 -crf 28 -preset veryfast -acodec aac compressed.mp4

    Resize Video

    ffmpeg -i input.mp4 -vf scale=1280:720 -c:a copy output_720p.mp4

    Merge Multiple Videos

    # Step 1: Create a file list
echo "file 'video1.mp4'" > file_list.txt
echo "file 'video2.mp4'" >> file_list.txt

# Step 2: Concatenate
ffmpeg -f concat -safe 0 -i file_list.txt -c copy merged.mp4

    Merge Multiple Audios

    echo "file 'audio1.mp3'" > audio_list.txt
echo "file 'audio2.mp3'" >> audio_list.txt
ffmpeg -f concat -safe 0 -i audio_list.txt -c copy merged.mp3

    Merge Audio and Video

    ffmpeg -i input.mp4 -i input.mp3 -c:v copy -c:a aac -strict experimental output.mp4

    Replace Audio in a Video

    ffmpeg -i input.mp4 -i new_audio.mp3 -map 0:v -map 1:a -c:v copy -shortest replaced.mp4

    Extract Frames

    ffmpeg -i input.mp4 -vf fps=1 frame_%04d.png

    Extracts 1 frame per second, named frame_0001.png, frame_0002.png, etc.

    Add Fade Effects

    Video (fade in for 3s, fade out starting at 27s for 3s):

    ffmpeg -i input.mp4 -vf "fade=t=in:st=0:d=3,fade=t=out:st=27:d=3" -c:a copy faded.mp4

    Audio:

    ffmpeg -i input.mp3 -af "afade=t=in:ss=0:d=3,afade=t=out:st=57:d=3" faded.mp3

    Crossfade Transition (2 Videos)

    ffmpeg -i video1.mp4 -i video2.mp4 -filter_complex "[0:v][1:v]xfade=transition=fade:duration=2:offset=4[v];  [0:a][1:a]acrossfade=d=2[a]" -map "[v]" -map "[a]" output.mp4

    img2pdf

    Converts images to PDF.

    Installation:

    sudo apt install img2pdf

    Syntax:

    img2pdf [options] <images...> -o <output.pdf>

    Examples:

    img2pdf image.jpg -o converted_img.pdf
img2pdf --pagesize A4 --output out.pdf *.jpg
img2pdf --border 2cm:2.5cm -o out.pdf *.jpg
img2pdf --fit shrink -S A4 -o out.pdf *.jpg
img2pdf --from-file list.txt -o out.pdf
img2pdf --dry-run *.jpg       # Validate without creating output

    yt-dlp

    Downloads videos and audio from YouTube and other sites.

    Installation:

    sudo wget https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp -O /usr/bin/yt-dlp
sudo chmod a+rx /usr/bin/yt-dlp

    Syntax:

    yt-dlp [options] <URL>

    Examples:

    yt-dlp <URL>                          # Download video
yt-dlp -x --audio-format mp3 <URL>    # Extract audio as MP3
yt-dlp -f best <URL>                  # Best quality available
yt-dlp -o "%(title)s.%(ext)s" <URL>   # Download playlist with original titles
yt-dlp -4 <URL>                       # Force IPv4
yt-dlp -U                             # Update yt-dlp

    7. Network & Remote Access

    ssh

    Connects to a remote server securely via SSH.

    Prerequisites: - Server Public IP Address - Server Username - Private SSH Key File (.key or .pem) - Firewall rules allowing SSH (port 22)

    Syntax:

    ssh -i <ssh-key-file> <username>@<public-ip>

    Example:

    ssh -i ssh-key-2025.key ubuntu@161.118.160.112

    Fix Permission Errors:

    chmod 600 <ssh-key-file>

    scp

    Securely copies files between local and remote machines over SSH.

    Syntax:

    scp -i <ssh-key> <local-file> <username>@<host>:<remote-path>

    Example:

    scp -i ssh-key.key report.pdf ubuntu@161.118.160.112:/home/ubuntu/docs/

    rsync

    Synchronizes files and directories between locations (local or remote). Faster than scp for large transfers.

    Syntax:

    rsync [flags] <source> <destination>

    Relevant Flags:

    Flag Description
    -a Archive mode (preserves permissions, timestamps, symlinks)
    -v Verbose
    -z Compress during transfer
    -e ssh Use SSH for remote transfer

    Examples:

    rsync -av /source/ /backup/
rsync -avz -e ssh /local/path/ user@remote_host:/remote/path/

    wget

    Downloads files from the web.

    Syntax:

    wget <URL>

    curl

    Transfers data from or to a server (supports many protocols).

    Syntax:

    curl [options] <URL>

    Examples:

    curl ifconfig.me              # Get public IPv4 address
curl loca.lt/mytunnelpassword # Get public IP / tunnel password
curl -O https://example.com/file.zip   # Download and save with original name

    lsof

    Lists open files and the processes that opened them. Useful for checking ports.

    Syntax:

    lsof -i:<port_number>

    Example:

    lsof -i:3000

    nmcli

    Network Manager command-line tool for controlling network connections.

    Examples:

    nmcli device show                 # Show full network info (IP, DNS, etc.)
nmcli device wifi show-password   # Display current Wi-Fi password

    hostname

    Displays or sets the system's hostname.

    Example:

    hostname -I                       # Display local IP address(es)

    resolvectl

    Checks DNS resolution status.

    Syntax:

    resolvectl status

    localtunnel / ngrok

    Exposes a local server to the public internet.

    localtunnel (easiest):

    9. PDF Toolkit

    pdftk

    The classic PDF toolkit for merging, splitting, rotating, and more.

    Installation:

    sudo apt install pdftk

    Operations:

    Task Command
    Merge PDFs pdftk file1.pdf file2.pdf cat output merged.pdf
    Split (burst) pdftk input.pdf burst output page_%02d.pdf
    Extract pages 2–4 pdftk input.pdf cat 2-4 output extracted.pdf
    Extract non-contiguous pdftk input.pdf cat 1 3 5 output selected.pdf
    Remove page 2 pdftk input.pdf cat 1 3-end output removed.pdf
    Rotate page 3 (90° CW) pdftk input.pdf cat 1-2 3E 4-end output rotated.pdf
    Rotate all (180°) pdftk input.pdf cat 1-endS output rotated_all.pdf
    Background watermark pdftk input.pdf background bg.pdf output watermarked.pdf
    Stamp overlay pdftk input.pdf stamp stamp.pdf output stamped.pdf
    Custom merge order pdftk A=doc1.pdf B=doc2.pdf cat A1-3 B2-4 A4-end output merged.pdf
    Repair PDF pdftk broken.pdf output fixed.pdf
    Unlock PDF pdftk input.pdf input_pw YOUR_PASSWORD output unlocked.pdf

    Rotation Codes: N = 0°, E = 90°, S = 180°, W = 270°

    qpdf

    A modern, robust alternative to pdftk, especially for encryption tasks.

    Decrypt a PDF:

    qpdf --password=YOUR_PASSWORD --decrypt input.pdf unlocked_output.pdf

    gs (Ghostscript)

    Pre-installed tool for PDF and PostScript manipulation.

    Compress PDF:

    gs -dSAFER -dBATCH -dNOPAUSE -sDEVICE=pdfwrite    -dCompatibilityLevel=1.4 -dPDFSETTINGS=/screen    -sOutputFile=compressed.pdf input.pdf

    Extract Pages:

    gs -sDEVICE=pdfwrite -dNOPAUSE -dBATCH -dSAFER    -dFirstPage=1 -dLastPage=1    -sOutputFile=extracted.pdf input.pdf

    pdfcrack

    Brute-force PDF password recovery.

    Installation:

    sudo apt install pdfcrack

    Syntax:

    pdfcrack -f input.pdf

    john (John the Ripper)

    Password cracking tool. Can be used with PDF hashes via pdf2john.

    pdf2john input.pdf > pdf.hash
john pdf.hash

    11. Text Processing & Search

    grep

    Searches for patterns in files using regular expressions.

    Syntax:

    grep [flags] <pattern> <file>

    Relevant Flags:

    Flag Description
    -r Recursive search through directories
    -i Case-insensitive search
    -c Count matching lines only
    -n Show line numbers
    -E Extended regex (enables | OR operator)

    Examples:

    grep 'error' output.log

    Output:

    [2025-07-09] error: connection timeout
[2025-07-09] error: file not found

    grep -i 'efield' forces.txt

    grep -n 'cow' breeds.csv

    Output:

    3:Angus,cow,Scotland
7:Holstein,cow,Netherlands

    grep -ri 'hello' /Documents

    grep -E 'id: |correct: ' file.dat

    awk

    Pattern scanning and processing language. Excellent for column-based data.

    Syntax:

    awk '{print $<column>}' <file>

    Example:

    awk '{print $1}' filename.dat

    Prints the first column of each line.

    sed (Stream Editor)

    Edits text in a stream or file non-interactively.

    Syntax:

    sed [flags] 's<delimiter><old><delimiter><new><delimiter>[g]' <file>

    Relevant Flags:

    Flag Description
    -i Edit in-place (modifies the file directly)
    -n Suppress automatic printing

    Examples:

    sed 's/pizza/burger/g' food.txt

    Replaces all occurrences of "pizza" with "burger".

    sed -i 's/^start.*/begin from 30 steps;/' begin.txt

    Replaces entire lines starting with "start".

    sed '3d' file.txt

    Deletes line 3 (output only; use -i to modify file).

    Tip: Omit -i to preview changes before applying them.

    tr

    Translates or deletes characters. Simpler than sed but less powerful.

    Syntax:

    tr "<old_chars>" "<new_chars>" < <file>

    Example:

    cat file.txt | tr "a-z" "A-Z"

    Converts lowercase to uppercase.

    perl

    Extremely powerful text manipulation, especially for complex regex and arithmetic.

    Syntax:

    perl -i -pe 's/old/new/g' file.txt

    Examples:

    perl -i -pe 's/src=".*?"/src="new"/g' file.txt

    Non-greedy replacement.

    perl -i -pe 's/(\d+)/$1 * 2/ge' file.txt

    Doubles all numbers in the file.

    perl -pe 's/(id:\s*)(\d+)/$1 . ($2 - 1)/e; s/(correct:\s*)(\d+)/$1 . ($2 + 1)/e' data.yaml | grep -E 'id: || correct:'

    Advanced arithmetic substitution with regex.

    head

    Displays the first N lines of a file.

    Syntax:

    head -n <count> <file>

    Example:

    head -n 20 log.txt

    tail

    Displays the last N lines of a file.

    Syntax:

    tail -n <count> <file>

    Example:

    tail -n 50 log.txt

    Follow mode (real-time updates):

    tail -f log.txt

    less

    Paginated file viewer (more powerful than more).

    Navigation: - ↑ / ↓ — Scroll line by line - Page Up / Page Down — Scroll by page - /pattern — Search forward - ?pattern — Search backward - q — Quit

    Syntax:

    less <file>

    cat

    Concatenates and displays file contents.

    Syntax:

    cat <file>

    Examples:

    cat file1.txt file2.txt > combined.txt

    13. VPN & Tunneling

    Cloudflare WARP

    A VPN service by Cloudflare.

    Installation:

    sudo apt update && sudo apt install curl
curl -fsSL https://pkg.cloudflareclient.com/pubkey.gpg |   sudo gpg --yes --dearmor --output /usr/share/keyrings/cloudflare-warp-archive-keyring.gpg

echo "deb [signed-by=/usr/share/keyrings/cloudflare-warp-archive-keyring.gpg]   https://pkg.cloudflareclient.com/ $(lsb_release -cs) main" |   sudo tee /etc/apt/sources.list.d/cloudflare-client.list

sudo apt-get update && sudo apt-get install cloudflare-warp

    Usage:

    warp-cli registration new      # Register
warp-cli connect               # Connect
warp-cli status                # Check status
warp-cli disconnect            # Disconnect
warp-cli --help                # Full help

    Verify Connection:

    curl https://www.cloudflare.com/cdn-cgi/trace/
